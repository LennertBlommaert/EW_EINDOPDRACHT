Instructions

You bring life to the world by playing music.

Each note creates a different object:

  • Q (G) - Raise mountains
  • W (A) - Create tree
  • E (B) - Create cloud
  • R (C) - Create rock
  • T (D) - Create mushroom
  • Y (E) - Create evergreen
  • U (F) - Create flower

If you stop playing, the created objects start disappearing. You have to keep playing to keep the world alive!

Playing major chords makes the world brighter, whilst playing minor chords makes it darker.

Your mouseposition on the screen influences the effect on the notes you play.

Shortcuts

  • Holding shift + vertical mouse positon influences the volume.
  • Holding shift + horizontal mouse positon influences the BPM.
